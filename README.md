# 🔳 Gerador de QR Code e Senhas com Node.js

Este projeto é uma aplicação desenvolvida em Node.js que permite gerar QR Codes e senhas seguras diretamente pelo terminal.

O usuário pode escolher entre duas funcionalidades principais:
- 📱 Gerar QR Code a partir de um link
- 🔐 Gerar senha aleatória

O projeto foi desenvolvido como desafio prático da **Jornada Para o Futuro - Node.js** da DIO, com o objetivo de aplicar conceitos fundamentais de programação backend e organização de código.

## 🚀 Tecnologias utilizadas

- Node.js
- JavaScript
- prompt
- chalk
- qrcode-terminal
